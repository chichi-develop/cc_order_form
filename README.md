# cc_order_form

## 環境

- 共通
  - docker
  - node.js
- Server
  - express
  - sequelize
- Client
  - react

## インストール、実行

```txt
$ git clone https://github.com/chichi-develop/cc_order_form.git

$ cd cd cc_order_form

$ docker-compose up -d --build
※Warningがいくつか出ます（対応中）

=> chromeで開く（Firefoxだと崩れます（対応中）
   http://localhost:8080/
```
